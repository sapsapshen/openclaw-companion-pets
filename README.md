# Companion Pets

> 多宠物字符动画陪伴小部件 — 纯浏览器原生能力，可注入任何网页。

---

## ✨ 特性

- 🐶 纯浏览器原生能力，无需安装额外插件即可运行
- 🦊 多种宠物提供选择，陪伴你的工作与娱乐
- 🐾 动画丰富，通过字符画呈现不同的宠物形态
- 🖥️ 轻松注入到包含 openclaw 等框架在内的现代页面中

---

## 🚀 快速开始

### 方法一：启动本地服服务（推荐）
1. 在项目根目录执行 `npm install`（如果需要）。
2. 执行 `npm start` 启动本地服务器。
3. 根据控制台提示（通常是 http://localhost:3000）在浏览器中访问目标页面，观察页面是否有宠物出现！

### 方法二：直接引入
你也可以直接将源码引入到你的前端项目中加载运行。
