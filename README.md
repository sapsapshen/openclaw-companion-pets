# Companion Pets (陪伴宠物小部件)

## 📌 项目简介

**Companion Pets** 是一个基于纯浏览器原生能力构建的「多宠物字符动画陪伴小部件」！可以无缝注入到各类网页（特别是 openclaw webchat 等聊天页面），在浏览器右下角生成一只可互动的 ASCII/Emoji 字符宠物，为您的浏览和工作带来一份趣味陪伴。

## 🚀 核心特性

- 🐰 **丰富的宠物状态**：内置多种字符动画宠物，拥有不同情绪变化（开心、饥饿、伤心、睡眠等）以及动画表现。
- 🎮 **点击互动与养成**：您可以点击宠物进行互动，喂食或安抚它，时刻保持它的活跃度！
- 🔌 **全形态发布**：提供两种运行与加载形态，满足不同用户习惯：
  - **UserScript (油猴脚本)**: 使用 companion-pets.user.js 一键在浏览器脚本管理器中运行。
  - **浏览器扩展**: 基于 Manifest V3 标准的纯净 Chrome/Edge 扩展（位于 extension/ 目录下）。
- 💬 **智能聊天输入监听**：智能监听聊天输入框动作。输入特殊指令会触发宠物回应（例如输入 /pet 呼叫宠物，输入 /pet reset 重置宠物数据）。
- ⚡ **零依赖架构**：纯 Vanilla JavaScript 编写，完全无需任何庞大的外部框架或运行库，性能开销极低。

## 📦 安装与使用指南

### 方法一：启动本地服服务（推荐）
1. 在项目根目录执行 `npm install`（如果需要）。
2. 执行 `npm start` 启动本地服务器。
3. 根据控制台提示（通常是 http://localhost:3000）在浏览器中访问目标页面，观察页面是否有宠物出现！

### 方法二：作为浏览器扩展运行 (Chrome/Edge)
1. 下载或克隆本仓库到本地。
2. 打开浏览器的扩展管理页面（Chrome请输入 chrome://extensions/，并在右上角开启 **开发者模式**）。
3. 点击 **加载已解压的扩展程序 (Load unpacked)** 按钮。
4. 选择并加载本仓库内的 extension 文件夹。
5. 扩展启动后，在目标聊天网页中宠物即会自动装载。

### 方法三：本地开发调试
项目内置了简单的本地服务器环境：
npm start

## 🐾 彩蛋与特殊指令

您可以利用网页中的聊天框发送指令，与小宠物触发更多隐藏交互：
- **/pet** - 确认宠物当前状态，宠物会激动地向你报道！
- **/pet reset** - 当宠物不幸饿死（前往汪星/喵星）或出现未知卡死状况时，发送此指令可以重置本地缓存并复活宠物小部件。
